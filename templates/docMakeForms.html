Adding search forms to your website
Adding a search form to your site is very simple. Just copy any of these search forms 
Default search form: |DIR|/wgindex.html
Simplified version: |DIR|/wgsimple.html
Bare query box: |DIR|/wgverysimple.html
into your web directory and place a link to it. You can edit this file cosmetically if you like, just leave the FORM tag and the INPUT tags the same. If you want fewer input tags, make the ones you don't want to appear TYPE="HIDDEN" and they will disappear from the screen.

To incorporate a simple search box into an existing page, just copy this code: 


	<FORM method=get ACTION="|WEBGLIMPSE|">
	<INPUT TYPE=HIDDEN NAME="ID" VALUE="|ID|">
	<INPUT TYPE=HIDDEN NAME="cache" VALUE="yes"> 
 	<INPUT TYPE=submit VALUE="Search: "><INPUT NAME=query size=30>
	</FORM>
This is a general search box to search the entire index, if you want neighborhood searching see'Adding neighborhood search boxes to your site'.

If you are using filters to index non-English text, or non-ascii files such as PDF or Word, you will also want to add the tag 

	<INPUT TYPE=HIDDEN NAME="nonascii" VALUE="yes">
